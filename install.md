# SideQuest — Project Requirements

A gamified real-world quest platform where users complete challenges, earn XP, level up, and compete on the leaderboard.

## 🛠️ Required Software & Tools

### Core Dependencies
- **Node.js** — v20 or higher (Recommended: v22 LTS) - https://nodejs.org/en/download Use the pre-built installation file (scroll down)
- **pnpm** — v9 or higher (package manager) - https://pnpm.io/installation
- **MySQL** — v8.0+ (or PlanetScale / any MySQL-compatible database)
- **Git**

### Recommended Development Tools
- **VS Code** + extensions:
  - ESLint
  - Prettier
  - Tailwind CSS IntelliSense
  - Docker (optional)

### Optional but Recommended
- **Docker** & **Docker Compose** (for local MySQL)
- **Redis** (future caching / real-time features)
- **S3-compatible storage** (Cloudflare R2, AWS S3, or local MinIO)
